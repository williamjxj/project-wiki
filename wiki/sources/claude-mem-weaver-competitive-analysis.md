# mem-weaver Competitive Analysis

## Key Claims

- Synthesis/compiler beats raw-history RAG for this product shape.
- Phase A absent → write-only pipeline, not production-ready as interactive product.
- Many items marked not built: semantic search, value gate, public LLM, _index.md.
- Differentiation: local-first + human-readable markdown + synthesized wiki.
- VC memory startup path discouraged vs Mem0/Zep/Letta.
- Strong for solo dev/Obsidian/privacy; weak for enterprise without auth.
- OSS primary; monetization via enterprise skip.

## Unique Insights

- Explicit don't-do-a-memory-startup framing with funding comparison.
